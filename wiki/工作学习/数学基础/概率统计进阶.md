## 一、随机变量与分布函数

### PDF、CDF、PMF 的关系

| 概念 | 符号 | 离散型 | 连续型 | 含义 |
|------|------|--------|--------|------|
| 概率质量函数 | PMF | P(X = x) | — | 单点的概率 |
| 概率密度函数 | PDF | — | f(x) | 密度，不是概率！f(x)dx 才是 |
| 累积分布函数 | CDF | F(x) = P(X ≤ x) | F(x) = ∫_{-∞}^{x} f(t)dt | 不超过 x 的概率 |

**关键区分**：连续型 PDF 在某点的值可以 > 1（比如 U(0, 0.5) 的 PDF = 2），但积分=1。PMF 的值永远 ≤ 1。

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# PDF vs CDF 的关系
x = np.linspace(-4, 4, 500)

# 标准正态分布
pdf_vals = stats.norm.pdf(x)
cdf_vals = stats.norm.cdf(x)

# CDF 是 PDF 的积分，PDF 是 CDF 的导数
# F(x) = ∫_{-∞}^{x} f(t) dt
# f(x) = dF/dx

# 验证: P(a < X ≤ b) = F(b) - F(a) = ∫_a^b f(x)dx
a, b = -1, 2
prob_from_cdf = stats.norm.cdf(b) - stats.norm.cdf(a)
# scipy 有直接积分方法
prob_from_integrate = stats.norm.cdf(b) - stats.norm.cdf(a)
print(f"P({a} < X ≤ {b}) = {prob_from_cdf:.4f}")

# PDF 在某点 > 1 的例子
uniform_pdf = stats.uniform.pdf(0.25, loc=0, scale=0.3)  # U(0, 0.3)
print(f"U(0,0.3) 的 PDF 在 x=0.25 处: {uniform_pdf:.2f} (可以 > 1)")

# 验证归一化: ∫ PDF dx = 1
from scipy.integrate import quad
integral, _ = quad(stats.norm.pdf, -np.inf, np.inf)
print(f"∫ PDF dx = {integral:.4f} (应该 = 1)")
```

### 分位数函数（PPF）

PPF = CDF 的反函数。Q(p) = 满足 P(X ≤ x) = p 的 x。

```python
# 分位数：CDF 的反函数
# 95% 置信区间的 z 值
z_975 = stats.norm.ppf(0.975)   # ≈ 1.96
z_025 = stats.norm.ppf(0.025)   # ≈ -1.96
print(f"P(|Z| < {z_975:.4f}) = 0.95")

# 验证：P(-1.96 < Z < 1.96) = 0.95
prob_95 = stats.norm.cdf(z_975) - stats.norm.cdf(z_025)
print(f"验证: {prob_95:.4f}")
```

## 二、多维随机变量

### 联合分布、边缘分布、条件分布

```
联合分布  p(x, y)：两个变量同时取某值的概率（密度）
边缘分布  p(x) = ∫ p(x, y) dy 或 Σ p(x, y)：只看一个变量
条件分布  p(y|x) = p(x, y) / p(x)：已知 x 时 y 的分布
```

```python
import numpy as np
from scipy import stats
import seaborn as sns

# 二维正态分布
mean = [0, 0]
cov = [[1.0, 0.7],   # 相关系数 0.7
       [0.7, 1.0]]

# 从联合分布采样
np.random.seed(42)
samples = np.random.multivariate_normal(mean, cov, size=5000)

# 联合分布：前 5 个样本
print("联合分布样本 (x, y) 前 5 个:")
print(samples[:5])

# 边缘分布：只看 x
x_samples = samples[:, 0]
print(f"\n边缘分布 X: mean={x_samples.mean():.3f}, std={x_samples.std():.3f}")

# 条件分布：给定 x > 1.5 时 y 的分布
condition = samples[:, 0] > 1.5
y_given_x_large = samples[condition, 1]
print(f"条件分布 P(Y | X > 1.5): mean(Y)={y_given_x_large.mean():.3f}")

# 注意：由于正相关(r=0.7)，X 大时 Y 也倾向于大
print(f"  无条件 Y 均值: {samples[:, 1].mean():.3f}")
print(f"  条件 Y 均值: {y_given_x_large.mean():.3f}（更大！）")
```

### 协方差矩阵

```
Cov(X, Y) = E[(X - E[X])(Y - E[Y])]

协方差矩阵 Σ =
[Var(X)   Cov(X,Y)]
[Cov(Y,X) Var(Y)  ]
```

**ML 中的应用**：
1. **PCA 降维**：对协方差矩阵做特征分解，取最大的 k 个特征值对应的特征向量作为主成分
2. **马氏距离**：用协方差矩阵的逆加权，消除各维度尺度的影响
3. **多元高斯分布**：Σ 决定分布的"形状"和"方向"

```python
# PCA 的本质 = 协方差矩阵的特征分解
from sklearn.decomposition import PCA

# 生成有相关性的数据
np.random.seed(42)
X_raw = np.random.randn(200, 10)  # 200 个样本，10 个特征
# 加一些相关性
X_raw[:, 2] = 0.8 * X_raw[:, 0] + 0.2 * np.random.randn(200)
X_raw[:, 3] = 0.6 * X_raw[:, 1] + 0.4 * np.random.randn(200)

# 手动 PCA（协方差矩阵 + 特征分解）
X_centered = X_raw - X_raw.mean(axis=0)
cov_matrix = np.cov(X_centered, rowvar=False)
eigenvalues, eigenvectors = np.linalg.eigh(cov_matrix)

# 按特征值降序排列
idx = np.argsort(eigenvalues)[::-1]
eigenvalues = eigenvalues[idx]
eigenvectors = eigenvectors[:, idx]

# 取前 k=2 个主成分
k = 2
X_pca_manual = X_centered @ eigenvectors[:, :k]

# 与 sklearn 对比
pca = PCA(n_components=2)
X_pca_sklearn = pca.fit_transform(X_raw)

print(f"前 5 个特征值: {eigenvalues[:5]}")
print(f"解释方差比例: {eigenvalues[:k].sum() / eigenvalues.sum():.2%}")
print(f"与 sklearn 结果一致: {np.allclose(np.abs(X_pca_manual), np.abs(X_pca_sklearn), atol=1e-10)}")
```

## 三、大数定律与中心极限定理（面试必考！）

### 大数定律 (LLN)

**直觉**：样本够多时，样本均值趋近于总体均值。

```python
# 大数定律的演示
np.random.seed(42)

# 从指数分布（偏态）中抽样，观察样本均值如何趋近总体均值
true_mean = 2.0  # 指数分布 λ=1/2 的期望
sample_sizes = [10, 50, 100, 500, 1000, 5000, 10000]

print("大数定律：样本均值 → 总体均值")
print(f"总体均值 = {true_mean}")
for n in sample_sizes:
    sample = np.random.exponential(scale=true_mean, size=n)
    sample_mean = sample.mean()
    error = abs(sample_mean - true_mean)
    print(f"  n={n:5d}: 样本均值={sample_mean:.4f}, 误差={error:.4f}")
# 随着 n 增大，误差越来越小
```

### 中心极限定理 (CLT)

**不管原始分布是什么，样本均值的分布趋近于正态分布。**

这是统计学中最惊人的结论——它让正态分布成为"万能分布"。

```python
# CLT 的深刻演示
np.random.seed(42)

def demo_clt(population_generator, pop_name, n_samples=2000, sample_size=30):
    """演示中心极限定理"""
    sample_means = []
    for _ in range(n_samples):
        sample = population_generator(sample_size)
        sample_means.append(sample.mean())

    sample_means = np.array(sample_means)
    # 理论值
    theoretical_mean = sample_means.mean()

    print(f"\n--- {pop_name} ---")
    print(f"  样本均值的均值: {sample_means.mean():.4f}")
    print(f"  样本均值的标准差 (标准误): {sample_means.std():.4f}")

    # 正态性检验
    from scipy.stats import shapiro, normaltest
    # 偏度和峰度
    from scipy.stats import skew, kurtosis
    print(f"  偏度: {skew(sample_means):.4f} (正态为 0)")
    print(f"  超额峰度: {kurtosis(sample_means):.4f} (正态为 0)")

# 从四种完全不同的分布抽样，看样本均值的分布
# 1. 指数分布（严重右偏）
demo_clt(lambda n: np.random.exponential(scale=2, size=n),
         "指数分布 (强右偏)")

# 2. 均匀分布
demo_clt(lambda n: np.random.uniform(0, 10, size=n),
         "均匀分布")

# 3. 二项分布
demo_clt(lambda n: np.random.binomial(10, 0.3, size=n),
         "二项分布")

# 4. 混合分布（两个峰）
def bimodal(n):
    half = n // 2
    return np.concatenate([
        np.random.normal(0, 1, half),
        np.random.normal(5, 1, n - half)
    ])

demo_clt(bimodal, "双峰混合分布")
# 所有四种情况的样本均值分布都趋近正态！
```

**CLT 的三个关键点**：

1. **对原始分布几乎没要求**：只要方差有限、独立同分布
2. **样本量多大够用？** 经验上 n ≥ 30。但偏态越严重需要 n 越大
3. **标准误**：σ/√n——样本量翻倍，标准误缩小 √2 倍，不是缩小一半

### 面试经典问题：CLT 的意义是什么？

**标准回答**：

1. CLT 让**置信区间**和**假设检验**成为可能——我们不需要知道原始分布，样本均值分布就是正态
2. **AB 测试**中，不管用户的 CTR 分布多奇怪，样本均值的差异都近似正态，所以可以用 z/t 检验
3. ML 中 **Bagging**（如随机森林）的有效性也可以用 CLT 解释：多棵树的均值比单棵树方差小
4. **局限性**：CLT 说的是"样本均值的分布"，不是原始数据的分布，数据本身不一定正态

## 四、参数估计 → 损失函数的基础

### 矩估计 (Method of Moments)

用样本矩代替总体矩，解出参数。

```python
# 矩估计示例：估计正态分布的两个参数
np.random.seed(42)
true_mu, true_sigma = 5.0, 2.0
data = np.random.normal(true_mu, true_sigma, size=1000)

# 矩估计：用样本均值估计 μ，用样本方差估计 σ²
mu_mom = data.mean()
sigma_mom = data.std(ddof=1)  # ddof=1 无偏估计

print("矩估计结果:")
print(f"  μ: 真实={true_mu}, 估计={mu_mom:.4f}")
print(f"  σ: 真实={true_sigma}, 估计={sigma_mom:.4f}")
```

### 极大似然估计 (MLE)

**核心思想**：选参数 θ，使得观察到当前数据的概率（似然）最大。

```
θ_MLE = argmax_θ L(θ|x) = argmax_θ Π p(x_i|θ)

通常会取对数：log-likelihood = Σ log p(x_i|θ)
```

### MLE 与损失函数的关系 —— 面试必考！

**平方损失 (MSE) ↔ 假设误差服从正态分布的 MLE**
**交叉熵损失 ↔ 假设标签服从伯努利/类别分布的 MLE**

```python
import numpy as np
from scipy.optimize import minimize

# === 示例1：用 MLE 估计正态分布的参数 ===
np.random.seed(42)
data = np.random.normal(loc=5.0, scale=2.0, size=500)

# 负对数似然函数（最小化这个等价于最大化似然）
def neg_log_likelihood_normal(params, data):
    mu, log_sigma = params
    sigma = np.exp(log_sigma)  # 保证 sigma > 0
    n = len(data)
    # 正态分布的负 log-likelihood
    nll = n * np.log(sigma * np.sqrt(2 * np.pi)) + np.sum((data - mu)**2) / (2 * sigma**2)
    return nll

# 优化找 MLE
result = minimize(neg_log_likelihood_normal, x0=[0, 0], args=(data,),
                  method='Nelder-Mead')
mu_mle = result.x[0]
sigma_mle = np.exp(result.x[1])
print(f"MLE: μ={mu_mle:.4f}, σ={sigma_mle:.4f}")

# === 示例2：MSE = 正态分布误差下的 MLE ===
# 线性回归 y = wx + b + ε, ε ~ N(0, σ²)
# 似然函数 ∝ exp(-Σ(y - wx - b)² / (2σ²))
# 最大化似然 ≡ 最小化 Σ(y - wx - b)² ≡ 最小化 MSE

# === 示例3：交叉熵 = 分类分布下的 MLE ===
# 二分类：y_i ~ Bernoulli(p_i)
# 似然 L = Π p_i^y_i × (1-p_i)^(1-y_i)
# 负对数似然 = -Σ [y_i log(p_i) + (1-y_i) log(1-p_i)]
# 这就是二元交叉熵！

def binary_cross_entropy(y_true, y_pred):
    """二元交叉熵 = 伯努利分布的负对数似然"""
    eps = 1e-15  # 防 log(0)
    y_pred = np.clip(y_pred, eps, 1 - eps)
    return -np.mean(y_true * np.log(y_pred) + (1 - y_true) * np.log(1 - y_pred))

# 验证：MLE 角度理解损失函数
y_true_test = np.array([1, 0, 1, 0, 1])
y_pred_good = np.array([0.9, 0.1, 0.8, 0.2, 0.95])   # 预测准
y_pred_bad = np.array([0.6, 0.4, 0.5, 0.6, 0.55])    # 预测不准

print(f"好的预测 BCE = {binary_cross_entropy(y_true_test, y_pred_good):.4f}")
print(f"差的预测 BCE = {binary_cross_entropy(y_true_test, y_pred_bad):.4f}")
```

### MLE vs MAP

| 方法 | 优化目标 | 核心思想 | 正则化联系 |
|------|---------|---------|-----------|
| MLE | argmax P(D\|θ) | 只看数据 | 无正则化 |
| MAP | argmax P(θ\|D) = argmax P(D\|θ) × P(θ) | 数据 + 先验 | **L1 先验=Laplace, L2 先验=Gaussian** |

```python
# MAP 与正则化的联系
# L2 正则 = 假设参数服从 N(0, 1/λ) 的先验
# L1 正则 = 假设参数服从 Laplace(0, 1/λ) 的先验

# 带 L2 正则的线性回归 = 高斯先验下的 MAP
# Loss = MSE + λ||w||² = -log p(y|X,w) - log p(w)  (忽略常数)
# 其中 p(w) ~ N(0, 1/λ)

# 这就是为什么 L2 也叫"权重衰减"：从贝叶斯视角，它在说"我相信参数应该接近 0"
```

**面试话术**："MLE 只看数据拟合，容易过拟合。MAP 引入了参数的先验分布，相当于正则化。L2 正则等价于高斯先验的 MAP，L1 正则等价于 Laplace 先验的 MAP。L1 产生稀疏解是因为 Laplace 分布在 0 处有尖峰，推动更多参数精确为 0。"

## 五、贝叶斯定理与贝叶斯推断

### 贝叶斯定理

```
P(θ|D) = P(D|θ) × P(θ) / P(D)

后验概率 = 似然 × 先验 / 证据
```

### 贝叶斯更新

贝叶斯方法的核心：**先验 + 新数据 → 后验**，后验又成为下一次更新的先验。

```python
import numpy as np
from scipy import stats

# 贝叶斯更新：Beta-Binomial 共轭
# 场景：估计硬币正面的概率 θ
# 先验：θ ~ Beta(α, β)
# 似然：观察到 k 次正面，n 次抛掷 ~ Binomial(n, θ)
# 后验：θ ~ Beta(α + k, β + n - k)  ← 共轭！简单加法

def bayesian_update_beta(alpha_prior, beta_prior, successes, failures):
    """
    Beta-Binomial 共轭更新
    先验 Beta(α,β) + 观察到 s 成功 f 失败 → 后验 Beta(α+s, β+f)
    """
    alpha_post = alpha_prior + successes
    beta_post = beta_prior + failures
    return alpha_post, beta_post

# 初始无信息先验 θ ~ Beta(1, 1) = Uniform(0, 1)
alpha, beta = 1, 1
print(f"先验: Beta({alpha}, {beta}), 期望={alpha/(alpha+beta):.3f}")

# 观察数据：10 次抛掷，7 次正面
alpha, beta = bayesian_update_beta(alpha, beta, successes=7, failures=3)
print(f"第1次更新后: Beta({alpha}, {beta}), 期望={alpha/(alpha+beta):.3f}")

# 再观察：20 次抛掷，15 次正面
alpha, beta = bayesian_update_beta(alpha, beta, successes=15, failures=5)
print(f"第2次更新后: Beta({alpha}, {beta}), 期望={alpha/(alpha+beta):.3f}")

# 后验均值和置信区间
posterior_mean = alpha / (alpha + beta)
# 95% 最高密度区间
ci_low = stats.beta.ppf(0.025, alpha, beta)
ci_high = stats.beta.ppf(0.975, alpha, beta)
print(f"\n最终后验: 均值={posterior_mean:.3f}")
print(f"95% 可信区间: [{ci_low:.3f}, {ci_high:.3f}]")
```

### 贝叶斯推断在 ML 中的应用

| 方法 | 贝叶斯视角 | 用途 |
|------|----------|------|
| 朴素贝叶斯 | 贝叶斯定理 + 特征独立假设 | 文本分类、垃圾邮件过滤 |
| 贝叶斯优化 | 高斯过程先验 + 观察 → 后验 | 超参数调优 |
| 变分推断 (VI) | 用简单分布 q 近似后验 p | VAE、大规模贝叶斯模型 |
| MCMC | 从后验分布中采样 | 贝叶斯 A/B 测试 |
| Thompson Sampling | 后验采样来做 explore/exploit | 推荐系统、广告投放 |

```python
# 朴素贝叶斯：最简单的贝叶斯分类器
from sklearn.naive_bayes import GaussianNB
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split

X, y = make_classification(n_samples=500, n_features=5, random_state=42)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

nb = GaussianNB()
nb.fit(X_train, y_train)
print(f"朴素贝叶斯准确率: {nb.score(X_test, y_test):.3f}")
print(f"类的先验概率 P(y): {nb.class_prior_}")
print(f"每个类的均值 θ_y: {nb.theta_}")
```

## 六、假设检验深入

基础概念（p 值、t 检验、AB 测试流程）见 [[工作学习/统计学基础/假设检验与AB测试|假设检验与AB测试]]，这里深入讲进阶主题。

### 第一类错误 vs 第二类错误

| — | H₀ 为真 | H₀ 为假 |
|---|---------|---------|
| 不拒绝 H₀ | 正确 ✓ | **第二类错误** (β) |
| 拒绝 H₀ | **第一类错误** (α) | 正确 ✓ (Power = 1-β) |

```
记忆口诀：
第一类错误 = 假阳性（没病说有病）→ α = 0.05
第二类错误 = 假阴性（有病说没病）→ β = power 分析
```

**α 和 β 的权衡**：降低 α（更严格的显著性），β 就会增大（更难检测真实效果）。这需要更大的样本量来同时控制两者。

### 统计功效 (Power) 分析

**Power = 1 - β** = 当实际上有差异时，能检测出来的概率。

```python
from scipy import stats
import numpy as np

def power_analysis(effect_size, n, alpha=0.05):
    """
    两样本 t 检验的功效计算
    effect_size: Cohen's d = (μ₁ - μ₀) / σ
    """
    # 非中心参数
    ncp = effect_size * np.sqrt(n / 2)
    # 临界值
    df = 2 * n - 2
    t_crit = stats.t.ppf(1 - alpha / 2, df)
    # Power = P(|t| > t_crit | H₁ 为真)
    power = 1 - stats.nct.cdf(t_crit, df, ncp) + stats.nct.cdf(-t_crit, df, ncp)
    return power

# 如果真实提升 0.2 个标准差，每组 100 个样本
power = power_analysis(effect_size=0.2, n=100)
print(f"小效果(0.2σ), n=100 的功效: {power:.2%}")  # 可能只有 20-30%
# 效果越小、需要的样本量越大
```

### p 值的争议

1. **p < 0.05 的二分法过于粗暴**：p=0.049 和 p=0.051 实际上差别很小
2. **p-hacking**：试多个检验、逐步增加样本量直到显著 → 膨胀第一类错误
3. **p 值不回答"效应有多大"**：大样本下 trivial 的效应也能 p < 0.05
4. **建议**：报告效应量和置信区间，不只报告 p 值

```python
# 演示 p-hacking 的危险
# 对一个完全没效果的处理，分成 20 个指标检验，纯碰运气
np.random.seed(42)
n_significant = 0
for metric in range(20):
    control = np.random.normal(100, 15, 500)
    treatment = np.random.normal(100, 15, 500)  # 完全没差别
    t_stat, p_val = stats.ttest_ind(treatment, control)
    if p_val < 0.05:
        n_significant += 1

print(f"20 个指标中碰巧'显著'的: {n_significant} 个 (期望≈1)")
print("这提醒我们：多重比较时要做校正（Bonferroni / FDR）！")
```

## 七、信息论基础 → 损失函数设计

### 自信息与熵

```
自信息: I(x) = -log p(x)     单位：bit（以 2 为底）或 nat（以 e 为底）
熵:      H(X) = -Σ p(x) log p(x) = E[I(X)]   不确定性的度量
```

```python
import numpy as np

def entropy(probs, base=2):
    """计算熵"""
    probs = np.array(probs)
    probs = probs[probs > 0]  # 去掉 0（0 log 0 = 0）
    return -np.sum(probs * np.log(probs) / np.log(base))

# 熵的直观理解
print(f"确定事件 p=[1,0] 熵 = {entropy([1, 0]):.2f}")      # 0
print(f"均匀 p=[0.5,0.5] 熵 = {entropy([0.5, 0.5]):.2f}")   # 最大 = 1
print(f"均匀 4 类 熵 = {entropy([0.25]*4):.2f}")            # 2
print(f"偏斜 p=[0.9,0.1] 熵 = {entropy([0.9, 0.1]):.2f}")   # 较小
```

### KL 散度

**衡量两个分布 P 和 Q 的"距离"**（不是真正的距离，不对称）。

```
D_KL(P || Q) = Σ P(x) log(P(x) / Q(x)) = E_P[log P - log Q]
```

```python
def kl_divergence(p, q, base=2):
    """KL(P||Q) —— 注意顺序！"""
    p, q = np.array(p), np.array(q)
    p, q = p[p > 0], q[p > 0]  # 简化处理
    # 只有 P(x) > 0 的项才需要算
    mask = p > 0
    return np.sum(p[mask] * np.log(p[mask] / q[mask]) / np.log(base))

# KL(P||Q) 的直观理解
p_uniform = np.array([0.5, 0.5])
q_skewed = np.array([0.9, 0.1])

# 用均匀分布去近似偏斜分布 → KL 大
print(f"KL(均匀||偏斜) = {kl_divergence(p_uniform, q_skewed):.3f}")
# 用偏斜分布去近似均匀分布 → KL 大（不同的惩罚）
print(f"KL(偏斜||均匀) = {kl_divergence(q_skewed, p_uniform):.3f}")
```

### 交叉熵

**交叉熵 = 熵 + KL 散度**

```
H(P, Q) = -Σ P(x) log Q(x) = H(P) + D_KL(P || Q)
```

**为什么用交叉熵做分类损失？**

```
最小化交叉熵 ≡ 最小化 KL(P||Q)（因为 H(P) 是常数）
             ≡ 让 Q（模型预测）尽可能接近 P（真实分布）
             ≡ MLE（因为交叉熵 = 分类分布的负对数似然）
```

```python
def cross_entropy(p, q, base=2):
    """H(P, Q) = -Σ P(x) log Q(x)"""
    p, q = np.array(p), np.array(q)
    mask = p > 0
    return -np.sum(p[mask] * np.log(q[mask]) / np.log(base))

# 验证：CE = Entropy + KL
p_true = np.array([1, 0, 0])     # 真实标签（独热编码）
q_pred_good = np.array([0.8, 0.1, 0.1])   # 预测不错
q_pred_bad = np.array([0.4, 0.3, 0.3])    # 预测不好

print(f"H(P) = {entropy(p_true):.3f}")
print(f"CE(P, Q_good) = {cross_entropy(p_true, q_pred_good):.3f}")
print(f"CE(P, Q_bad) = {cross_entropy(p_true, q_pred_bad):.3f}")
# 交叉熵越小 → 预测越准

# 验证: CE = H(P) + KL(P||Q)
for name, q in [("good", q_pred_good), ("bad", q_pred_bad)]:
    ce = cross_entropy(p_true, q)
    h = entropy(p_true)
    kl = kl_divergence(p_true, q)
    print(f"Q_{name}: CE={ce:.3f}, H+KL={h+kl:.3f}, 匹配: {np.isclose(ce, h+kl)}")
```

### 信息论在 ML 中的应用汇总

| 概念 | ML 应用 |
|------|--------|
| 熵 | 决策树分裂标准（信息增益 = 分裂前熵 - 分裂后加权熵） |
| 交叉熵 | **分类问题默认损失函数**（PyTorch: `nn.CrossEntropyLoss`） |
| KL 散度 | VAE 的正则化项、知识蒸馏、衡量两个分布差异 |
| 互信息 | 特征选择（选与目标变量互信息大的特征）、InfoNCE 损失 |
| 困惑度 (Perplexity) | NLP 语言模型的评估指标：PPL = 2^H |

```python
# 互信息：一个变量包含关于另一个变量的多少信息
# I(X; Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)

from sklearn.feature_selection import mutual_info_classif
from sklearn.datasets import make_classification

X, y = make_classification(n_samples=500, n_features=10,
                            n_informative=3,  # 只有 3 个特征真正有用
                            random_state=42)

mi_scores = mutual_info_classif(X, y, random_state=42)
for i, score in enumerate(mi_scores):
    marker = " ★" if score > 0.05 else ""
    print(f"特征{i:2d}: 互信息={score:.4f}{marker}")
# 有用的特征互信息明显更高
```

## 八、面试高频考点

### Q1: MLE 和 MAP 的区别？

A: MLE 最大化 P(D|θ)，只看数据；MAP 最大化 P(θ|D) ∝ P(D|θ)P(θ)，引入了参数的先验。MAP 等价于正则化——L2 正则 = 高斯先验，L1 正则 = Laplace 先验。数据少时 MAP 更好（先验起约束作用），数据无穷时两者等价（先验的影响被稀释）。

### Q2: 为什么分类用交叉熵而不是 MSE？

A: (1) 交叉熵 + softmax/sigmoid 的梯度形式简单，不会像 MSE 那样在预测接近 0 或 1 时梯度消失。(2) 交叉熵是分类分布的自然负对数似然，有概率解释。(3) MSE 对概率值施加"距离相等"的假设，但 0.9→0.8 和 0.5→0.4 的"错误严重程度"不同。

```python
# 演示：MSE + sigmoid 梯度消失 vs 交叉熵
def sigmoid(x):
    return 1 / (1 + np.exp(-x))

# 假设真实标签 y=1，logits=z（sigmoid 之前）
z = np.linspace(-5, 5, 100)

# MSE: L = (sigmoid(z) - 1)^2
# dL/dz = 2(sigmoid(z)-1) * sigmoid(z) * (1-sigmoid(z))
s = sigmoid(z)
dMSE = 2 * (s - 1) * s * (1 - s)

# BCE: L = -log(sigmoid(z))
# dL/dz = sigmoid(z) - 1
dBCE = s - 1

# 在 z 很负（预测接近 0，完全错误）的时候：
print(f"z=-5: |dMSE/dz| = {abs(dMSE[0]):.6f}")
print(f"z=-5: |dBCE/dz| = {abs(dBCE[0]):.6f}")
# BCE 的梯度大得多 → 学得更快
```

### Q3: 中心极限定理的意义和局限？

A: 意义在于为统计推断提供基础——无论原始分布多奇怪，样本均值的分布近似正态，所以可以用正态分布方法做置信区间和检验。局限：(1) 只适用于样本均值，不适用于中位数、极值等统计量。(2) 要求有限方差，柯西分布等厚尾分布不满足。(3) 独立性假设被破坏时（时间序列自相关），CLT 不成立。

### Q4: 如何向非技术人员解释 p 值和置信区间？

p 值："如果新算法实际上没效果，我们看到这个结果的概率是多少？如果概率很小（< 5%），那可能新算法确实有效果。"

置信区间："我们估计提升效果是 3%，但真实效果可能在 1% 到 5% 之间。如果整个区间都在 0 以上（比如 1-5%），那就是显著的。"

### Q5: 熵、交叉熵、KL 散度三者关系？

A: 交叉熵 = 熵 + KL 散度。H(P, Q) = H(P) + D_KL(P||Q)。熵 H(P) 是真实分布本身的不确定性（常数）。训练模型时最小化交叉熵 = 最小化 KL 散度 = 让 Q 逼近 P。
