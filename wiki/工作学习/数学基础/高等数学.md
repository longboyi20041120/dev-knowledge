# 高等数学

面向算法岗面试的高等数学——不是数学系的严谨证明，而是**机器学习中用到的那些数学**。

学完这篇，你能回答：梯度下降为什么沿负梯度方向？XGBoost 为什么用二阶泰勒？反向传播的数学本质是什么？

## 一、函数与极限

### 极限的直觉理解

极限描述的是"当 x 无限接近某个点时，f(x) 无限接近什么值"。ε-δ 定义是数学系的严谨表述，面试不需要背，但**理解它的精神**很重要：

> 对于任意小的 ε > 0，总存在 δ > 0，使得当 0 < |x - x₀| < δ 时，|f(x) - L| < ε。

翻译成人话：**你给我一个任意小的误差范围（ε），我就能找到一个足够靠近 x₀ 的区间（δ），在这个区间里函数值离 L 的差距不超过 ε。**

```python
import numpy as np

# 直观理解极限：x → 0 时，sin(x)/x → 1
x_values = [1.0, 0.5, 0.1, 0.01, 0.001, 0.0001]
for x in x_values:
    ratio = np.sin(x) / x
    print(f"x={x:.4f}, sin(x)/x={ratio:.6f}")
# x=1.0000, sin(x)/x=0.841471
# x=0.1000, sin(x)/x=0.998334
# x=0.0010, sin(x)/x=0.999999
# 越靠近 0，值越接近 1 → 极限为 1
```

### 极限的四则运算

如果 lim f(x) = A，lim g(x) = B，则：

| 运算 | 结果 |
|------|------|
| lim (f ± g) | A ± B |
| lim (f × g) | A × B |
| lim (f / g) | A / B（B ≠ 0） |
| lim (c × f) | c × A |

### 两个重要极限

```
1. lim(x→0) sin(x)/x = 1
2. lim(x→∞) (1 + 1/x)^x = e ≈ 2.71828
```

**第二个极限在 ML 中的意义**：e 是自然对数的底，交叉熵损失、softmax、sigmoid 中到处都是 e。理解 e 的来源有助于理解为什么这些函数长这样。

### 无穷小与无穷大

- **高阶无穷小**：x → 0 时，x² 比 x 更快趋近 0。在泰勒展开中，高阶无穷小是可以被截断的。
- **等价无穷小替换**：x → 0 时，sin x ~ x，tan x ~ x，ln(1+x) ~ x，e^x - 1 ~ x。在求极限时非常实用。

```python
# 验证等价无穷小
import math

x = 0.001
print(f"sin(x) = {math.sin(x):.8f}, x = {x:.8f}")       # sin(x) ≈ x
print(f"ln(1+x) = {math.log(1+x):.8f}, x = {x:.8f}")    # ln(1+x) ≈ x
print(f"e^x-1 = {math.exp(x)-1:.8f}, x = {x:.8f}")      # e^x - 1 ≈ x
```

### 函数的连续性

函数 f(x) 在 x₀ 处连续，意味着：lim(x→x₀) f(x) = f(x₀)。

**ML 视角**：我们优化的损失函数必须是连续的，否则梯度不存在，无法做梯度下降。ReLU 在 x=0 处不可导（但实践中用次梯度解决），这就是为什么后来有了 GELU、Swish 等平滑激活函数。

## 二、导数与微分

### 导数的定义

```
f'(x₀) = lim(h→0) [f(x₀+h) - f(x₀)] / h
```

**几何意义**：函数在 x₀ 处的切线斜率。
**物理意义**：瞬时变化率。
**ML 意义**：损失函数对参数的敏感度——参数调一点，损失变多少？

```python
import sympy as sp

# 用 sympy 求导
x = sp.Symbol('x')
f = x**3 + 2*x**2 - 5*x + 1
f_prime = sp.diff(f, x)
print(f"f(x) = {f}")
print(f"f'(x) = {f_prime}")
# f'(x) = 3*x**2 + 4*x - 5

# 数值求导验证
import numpy as np

def numerical_derivative(f, x, h=1e-6):
    """中心差分法求数值导数"""
    return (f(x + h) - f(x - h)) / (2 * h)

def f_np(x):
    return x**3 + 2*x**2 - 5*x + 1

x0 = 2.0
exact = 3*x0**2 + 4*x0 - 5  # 精确值
approx = numerical_derivative(f_np, x0)
print(f"精确导数 f'(2) = {exact}")
print(f"数值近似 f'(2) = {approx:.6f}")
```

### 链式法则 → 反向传播的数学基础

```
如果 y = f(u), u = g(x)，则 dy/dx = dy/du × du/dx
```

**这就是反向传播的全部数学！** 多层神经网络就是多重嵌套函数的链式求导。

```python
# 手动演示反向传播：一个简单神经元的梯度计算
# 前向：z = wx + b, a = sigmoid(z), L = (a - y)^2

import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def sigmoid_derivative(x):
    s = sigmoid(x)
    return s * (1 - s)

# 输入
x, y = 0.5, 1.0    # 输入和标签
w, b = 0.2, 0.1    # 初始参数

# === 前向传播 ===
z = w * x + b       # z = 0.2 * 0.5 + 0.1 = 0.2
a = sigmoid(z)      # a ≈ 0.550
loss = (a - y)**2   # L ≈ 0.202

# === 反向传播（链式法则） ===
# dL/da = 2(a - y)
dL_da = 2 * (a - y)                     # ≈ -0.900
# da/dz = sigmoid'(z) = a(1-a)
da_dz = sigmoid_derivative(z)           # ≈ 0.247
# dz/dw = x, dz/db = 1
dz_dw = x                               # = 0.5
dz_db = 1                               # = 1

# dL/dw = dL/da × da/dz × dz/dw
dL_dw = dL_da * da_dz * dz_dw
# dL/db = dL/da × da/dz × dz/db
dL_db = dL_da * da_dz * dz_db

print(f"损失: {loss:.4f}")
print(f"梯度 dL/dw: {dL_dw:.4f}")  # ≈ -0.111
print(f"梯度 dL/db: {dL_db:.4f}")  # ≈ -0.223

# 用 PyTorch 验证
import torch
w_t, b_t = torch.tensor([0.2], requires_grad=True), torch.tensor([0.1], requires_grad=True)
x_t, y_t = torch.tensor([0.5]), torch.tensor([1.0])
z_t = w_t * x_t + b_t
a_t = torch.sigmoid(z_t)
loss_t = (a_t - y_t)**2
loss_t.backward()
print(f"PyTorch 验证 dL/dw: {w_t.grad.item():.4f}")  # 应与上面一致
print(f"PyTorch 验证 dL/db: {b_t.grad.item():.4f}")
```

**面试话术**："反向传播的本质就是链式法则。神经网络是多层复合函数，从输出层开始逐层往回算梯度，每层用上一层的梯度乘以本层激活函数的导数。这就是 dL/dw_i = dL/d_output × ... × d(layer_i)/dw_i。"

### 高阶导数

- **一阶导数**：变化率（速度）
- **二阶导数**：变化率的变化率（加速度），也反映函数的**凹凸性**
- **ML 应用**：二阶导数（Hessian 矩阵）用于牛顿法优化、XGBoost 的二阶泰勒展开

## 三、中值定理 → 优化理论的基石

### 三大中值定理对比

| 定理 | 条件 | 结论 | ML 关联 |
|------|------|------|---------|
| 罗尔定理 | f 在 [a,b] 连续，(a,b) 可导，f(a)=f(b) | 存在 ξ 使 f'(ξ)=0 | 导数为零可能是极值点 |
| 拉格朗日 | f 在 [a,b] 连续，(a,b) 可导 | 存在 ξ 使 f'(ξ)=(f(b)-f(a))/(b-a) | **梯度下降收敛性的理论基础** |
| 柯西 | f,g 在 [a,b] 连续，(a,b) 可导，g'(x)≠0 | 存在 ξ 使 f'(ξ)/g'(ξ)=(f(b)-f(a))/(g(b)-g(a)) | 洛必达法则的证明基础 |

```python
import numpy as np
import matplotlib.pyplot as plt

# 可视化拉格朗日中值定理
def f(x):
    return x**3 - 3*x**2 + 2

a, b = 0, 3
slope = (f(b) - f(a)) / (b - a)
print(f"区间 [{a}, {b}] 上的平均变化率: {slope:.4f}")
print(f"拉格朗日定理保证：存在某点的切线斜率 = {slope:.4f}")

# f'(x) = 3x² - 6x，解 f'(x) = slope
# 3x² - 6x - slope = 0
# 实际上 3x² - 6x = slope 的解就是满足定理的点
```

### 洛必达法则

当极限是 0/0 或 ∞/∞ 型时：lim f(x)/g(x) = lim f'(x)/g'(x)

**ML 中的应用**：理论上不多，但数学推导中常用。

## 四、泰勒展开 → XGBoost 的核心

### 泰勒公式

```
f(x) ≈ f(a) + f'(a)(x-a) + f''(a)(x-a)²/2! + f'''(a)(x-a)³/3! + ...
```

**核心思想**：用多项式近似任意函数。阶数越高，近似越准。

```python
import sympy as sp
import numpy as np

# 用 sympy 做泰勒展开
x = sp.Symbol('x')

# e^x 在 x=0 处的 5 阶泰勒展开
f_exp = sp.exp(x)
taylor_exp = sp.series(f_exp, x, 0, 6).removeO()  # 去掉 O(x^6) 余项
print(f"e^x 的 5 阶泰勒展开: {taylor_exp}")
# 1 + x + x**2/2 + x**3/6 + x**4/24 + x**5/120

# sin(x) 在 x=0 处的泰勒展开
f_sin = sp.sin(x)
taylor_sin = sp.series(f_sin, x, 0, 7).removeO()
print(f"sin(x) 的 6 阶泰勒展开: {taylor_sin}")
# x - x^3/6 + x^5/120

# 可视化：原函数 vs 不同阶数的泰勒近似
import matplotlib.pyplot as plt

x_vals = np.linspace(-3, 3, 200)
y_true = np.exp(x_vals)

# 不同阶数近似
y_0 = np.ones_like(x_vals)                          # 0 阶：常数
y_1 = 1 + x_vals                                     # 1 阶：线性
y_2 = 1 + x_vals + x_vals**2 / 2                    # 2 阶：二次
y_3 = 1 + x_vals + x_vals**2/2 + x_vals**3/6         # 3 阶

# 在 x=0 附近，低阶近似已经很准；远离 x=0 则需要更高阶
print(f"x=1: 真实={np.exp(1):.4f}, 1阶={y_1[166]:.4f}, 3阶={y_3[166]:.4f}")
print(f"x=3: 真实={np.exp(3):.4f}, 1阶={y_1[-1]:.4f}, 3阶={y_3[-1]:.4f}")
```

### XGBoost 为什么用二阶泰勒？

XGBoost 的目标函数在每一步迭代中，不是直接优化原损失函数，而是对损失函数做**二阶泰勒展开**后优化：

```
Loss ≈ Σ [L(y, ŷ_old) + g_i × f_new(x_i) + (1/2) × h_i × f_new(x_i)²] + Ω(f_new)
```

其中 g_i 是一阶梯度，h_i 是二阶梯度（Hessian）。

**为什么用二阶？**
1. **一阶只能线性近似**，二阶能捕捉曲率信息 → 收敛更快
2. **二阶展开后目标函数是二次型**，可以解析求解最优分裂点，不需要迭代优化
3. 传统 GBDT 只用一阶梯度（梯度下降），XGBoost 用二阶相当于**牛顿法**的思想

```python
# 演示：一阶近似 vs 二阶近似
import numpy as np

def f(x):
    """待近似的函数：f(x) = -ln(x) + (x-2)^2"""
    return -np.log(x) + (x - 2)**2

# 在 x0=1.5 处展开
x0 = 1.5
g = -1/x0 + 2*(x0 - 2)    # 一阶导数 f'(x0)
h = 1/x0**2 + 2             # 二阶导数 f''(x0)

# 近似
x_test = np.linspace(0.5, 3, 100)
f_true = f(x_test)
f_first = f(x0) + g * (x_test - x0)                     # 一阶近似
f_second = f(x0) + g * (x_test - x0) + 0.5 * h * (x_test - x0)**2  # 二阶近似

# 找近似的极小值
# 一阶近似：直线，没有极小值（梯度的方向）
# 二阶近似：抛物线，极小值在 x = x0 - g/h
x_opt = x0 - g / h
print(f"真实极小值约在 x≈1.0")
print(f"一阶梯度 g={g:.4f}，指向下降方向")
print(f"二阶给出的最优步长: -g/h = {-g/h:.4f}，新点 x={x_opt:.4f}")
print(f"牛顿法一步就能接近极小值！")
```

### 泰勒展开的 ML 应用汇总

| 应用 | 几阶展开 | 用途 |
|------|---------|------|
| XGBoost | 二阶 | 目标函数近似为二次型，解析求解 |
| 梯度下降 | 一阶 | 用一阶梯度决定下降方向 |
| 牛顿法 | 二阶 | 同时用梯度和 Hessian 决定方向和步长 |
| Fisher 信息矩阵 | 二阶 | 参数估计的精度下界（与 Hessian 期望相关） |
| 损失函数光滑化 | 任意 | 用多项式替代不可导函数 |

## 五、不定积分与定积分

### 牛顿-莱布尼茨公式

```
∫[a,b] f(x)dx = F(b) - F(a)，其中 F'(x) = f(x)
```

**定积分 = 面积**，不定积分 = 原函数族（差一个常数 C）。

```python
import sympy as sp

x = sp.Symbol('x')

# 不定积分
f = x**2 + 3*x + 2
F = sp.integrate(f, x)
print(f"∫({f})dx = {F} + C")
# x**3/3 + 3*x**2/2 + 2*x + C

# 定积分：∫[0,2] (x² + 3x + 2) dx
result = sp.integrate(f, (x, 0, 2))
print(f"∫[0,2] ({f})dx = {result}")
# 计算结果

# 数值积分（当解析解不好求时）
import numpy as np
from scipy import integrate

def f_np(x):
    return x**2 + 3*x + 2

result_num, error = integrate.quad(f_np, 0, 2)
print(f"数值积分结果: {result_num:.6f}，误差估计: {error:.2e}")
```

### ML 中的积分应用

1. **概率密度函数归一化**：∫ PDF dx = 1
2. **期望计算**：E[X] = ∫ x·p(x)dx（连续型）
3. **ROC 曲线下面积 AUC**：本质是定积分
4. **边缘分布**：p(x) = ∫ p(x,y)dy

```python
# AUC 本质是定积分的数值近似
import numpy as np
from sklearn.metrics import roc_auc_score

# 假数据
np.random.seed(42)
y_true = np.array([0, 0, 1, 1, 0, 1, 0, 1])
y_score = np.array([0.1, 0.3, 0.4, 0.35, 0.8, 0.6, 0.2, 0.9])

# AUC = 梯形法则对 ROC 曲线的数值积分
auc = roc_auc_score(y_true, y_score)
print(f"AUC = {auc:.4f}（本质是 ROC 曲线下的定积分）")
```

## 六、多元函数微分 → 梯度下降

### 偏导数

对多元函数 f(x₁, x₂, ..., xₙ)，偏导数 ∂f/∂xᵢ 表示**只改变 xᵢ，其它变量固定时** f 的变化率。

### 梯度

```
∇f = (∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ)
```

**梯度是一个向量，指向函数值增长最快的方向。**

这也是为什么梯度下降沿着**负梯度**方向走——梯度指向上山最快的方向，负梯度指向下山最快的方向。

```python
import numpy as np

# 演示：二元函数 f(x,y) = x² + 2y² 的梯度场
def f(x, y):
    return x**2 + 2*y**2

def gradient(x, y):
    """∇f = (2x, 4y)"""
    return np.array([2*x, 4*y])

# 梯度下降可视化
def gradient_descent_2d(start, lr=0.1, steps=10):
    path = [start]
    point = np.array(start, dtype=float)

    for _ in range(steps):
        grad = gradient(point[0], point[1])
        point = point - lr * grad  # 负梯度方向
        path.append(point.copy())

    return np.array(path)

start = (3.0, 2.0)
path = gradient_descent_2d(start, lr=0.3, steps=10)

print("梯度下降路径（负梯度方向）：")
for i, p in enumerate(path):
    print(f"  步骤{i}: ({p[0]:.4f}, {p[1]:.4f}), f={f(p[0], p[1]):.4f}")
print(f"  极小值点: (0, 0)，f(0,0)=0")
```

### 方向导数

沿方向 **v**（单位向量）的变化率 = ∇f · v（梯度和方向的点积）。

**梯度方向是方向导数最大的方向**，因为 ∇f · v ≤ ‖∇f‖ × ‖v‖ = ‖∇f‖，等号成立时 v 与 ∇f 同向。

### 面试核心问题：梯度下降为什么沿负梯度方向？

**标准回答**：

1. 梯度指向函数值**增长最快**的方向（方向导数最大）
2. 我们要**最小化**损失函数，所以沿**反方向**走
3. 从泰勒展开角度：f(x - η∇f) ≈ f(x) - η‖∇f‖²，当 η 足够小时，函数值**保证减小**
4. 这是最速下降法（Steepest Descent）

### Hessian 矩阵

```python
# Hessian 矩阵：二阶偏导数矩阵
import sympy as sp

x, y = sp.symbols('x y')
f_sym = x**2 + 2*x*y + 3*y**2

# 用 sympy 计算 Hessian
# ∂²f/∂x²  ∂²f/∂x∂y
# ∂²f/∂y∂x ∂²f/∂y²
hessian = sp.hessian(f_sym, (x, y))
print(f"Hessian 矩阵:\n{hessian}")
# [[2, 2], [2, 6]]

# Hessian 用于判断极值点类型：
# 正定 → 极小值点；负定 → 极大值点；不定 → 鞍点
```

**ML 中的 Hessian**：牛顿法用 Hessian 的逆矩阵来调整下降方向，使得优化不受参数尺度影响（但计算 Hessian 逆很昂贵，所以深度学习很少直接用）。

## 七、重积分

### 二重积分

二重积分 = 曲顶柱体的体积。在概率统计中用于计算二维随机变量的概率。

```python
from scipy import integrate

# 计算二重积分 ∬[0,1]×[0,1] (x² + y²) dx dy
def inner(y):
    return integrate.quad(lambda x: x**2 + y**2, 0, 1)[0]

result = integrate.quad(inner, 0, 1)[0]
print(f"∬(x²+y²)dxdy 在 [0,1]×[0,1] = {result:.6f}")

# 二重积分验证联合概率密度的归一化
# p(x,y) = 6xy²，x∈[0,1], y∈[0,1]
def inner_pdf(y):
    return integrate.quad(lambda x: 6 * x * y**2, 0, 1)[0]

normalization = integrate.quad(inner_pdf, 0, 1)[0]
print(f"联合概率密度归一化检查: {normalization:.6f}（应该=1）")
```

### 三重积分

三重积分 = 空间区域的"超体积"。在 ML 中主要出现在概率模型的归一化常数计算中（如多元高斯分布的归一化因子）。

## 八、级数

### 幂级数

```
∑ a_n × x^n
```

泰勒级数就是幂级数的一种。幂级数有**收敛半径**——超过这个半径，级数发散。

**ML 关联**：核方法（如 RBF 核）可以看作无穷维特征空间的幂级数展开。

### 傅里叶级数

**核心思想**：任何周期函数都可以表示为正弦和余弦波的叠加。

```
f(x) = a₀/2 + Σ[a_n cos(nx) + b_n sin(nx)]
```

```python
import numpy as np

# 演示：用傅里叶级数近似方波
def square_wave_fourier(x, n_terms):
    """用前 n_terms 项傅里叶级数近似方波"""
    result = np.zeros_like(x)
    for k in range(1, n_terms + 1, 2):  # 只有奇数项
        result += (4 / (np.pi * k)) * np.sin(k * x)
    return result

x = np.linspace(-np.pi, np.pi, 500)
true = np.sign(np.sin(x))  # 真实方波
approx_1 = square_wave_fourier(x, 1)   # 1 项
approx_5 = square_wave_fourier(x, 11)  # 前 6 个奇数项

# 项数越多，近似越好（但 Gibbs 现象：跳变处永远有 overshoot）
```

**ML 关联**：
1. **傅里叶特征**：Transformer 中的位置编码可以用傅里叶基函数（sinusoidal position encoding）
2. **频域分析**：卷积层的傅里叶解释，在频域做乘积等价于时域的卷积
3. **时序分析**：傅里叶变换用于提取周期性特征

```python
# Transformer 的正弦位置编码
def sinusoidal_position_encoding(seq_len, d_model):
    """Transformer 原文的位置编码"""
    position = np.arange(seq_len)[:, np.newaxis]
    div_term = np.exp(np.arange(0, d_model, 2) * (-np.log(10000.0) / d_model))
    pe = np.zeros((seq_len, d_model))
    pe[:, 0::2] = np.sin(position * div_term)   # 偶数维度用 sin
    pe[:, 1::2] = np.cos(position * div_term)    # 奇数维度用 cos
    return pe

pe = sinusoidal_position_encoding(10, 8)
print(f"位置编码形状: {pe.shape}")  # (10, 8)
print(f"位置 0: {pe[0, :4]}...")
print(f"位置 5: {pe[5, :4]}...")
```

## 九、面试高频考点

### Q1: 梯度下降为什么沿负梯度方向？

A: 梯度指向函数增长最快的方向（方向导数取最大值时与梯度同向）。要最小化损失函数，必须沿反方向。从一阶泰勒展开看：f(x - η∇f) ≈ f(x) - η‖∇f‖²，只要 η > 0，函数值保证下降。

### Q2: 泰勒展开在 ML 中的应用？

A: (1) XGBoost 用二阶泰勒展开近似损失函数，将优化问题转化为二次型，可以解析求解最优分裂点。(2) 梯度下降本身基于一阶泰勒展开。(3) 神经网络剪枝中的二阶方法用 Hessian 的泰勒展开近似评估权重重要性。

### Q3: 为什么激活函数要可导？

A: 反向传播需要计算梯度。如果激活函数不可导（如阶跃函数），梯度处处为零或不存在，网络无法训练。ReLU 在 x=0 处不可导，但实践中用次梯度（0 或 1）作为替代。

### Q4: 牛顿法和梯度下降的本质区别？

A: 梯度下降用**一阶泰勒展开**近似，方向是负梯度，步长固定。牛顿法用**二阶泰勒展开**近似，方向是 -H⁻¹∇f（Hessian 逆乘梯度），同时调整方向和步长。牛顿法收敛更快但计算 Hessian 逆太贵，适合小规模问题。

### Q5: Sigmoid 和 tanh 为什么会有梯度消失？

A: 从导数看，sigmoid'(x) = sigmoid(x)(1-sigmoid(x))，最大值约 0.25。多层连乘后梯度指数衰减，深层网络几乎收不到梯度信号。

```python
# 直观演示梯度消失
def sigmoid_grad(x):
    s = 1 / (1 + np.exp(-x))
    return s * (1 - s)

# 10 层 sigmoid，梯度连乘
grad = 1.0
for _ in range(10):
    grad *= sigmoid_grad(0)  # x=0 时导数最大约 0.25
print(f"10 层 sigmoid 后梯度衰减到: {grad:.8f}")
# 约 0.25^10 ≈ 9.5e-7 —— 几乎消失了
```
