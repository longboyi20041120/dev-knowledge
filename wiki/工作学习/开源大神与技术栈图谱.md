## 二、开源影响力分级

**S 级（改变整个软件行业）**
Bjarne Stroustrup（C++）· Matt Mullenweg（WordPress）· John Resig（jQuery）· Robert Griesemer（Go）· Fabrice Bellard（FFmpeg/QEMU）

**A 级（改变某个技术领域）**
Rich Harris（Svelte）· TJ Holowaychuk（Express/Koa）· Isaac Z. Schlueter（npm）· Scott Chacon（GitHub/Git）

**B 级（大型项目核心贡献者）**
Mathias Bynens · Thomas Dohmke · Xinchen Hui · Ben Haynes

## 三、按技术栈看：一张"学习路线"地图

```
C++          → Bjarne Stroustrup        → STL、模板、RAII      → 游戏引擎/高频交易/操作系统
Go           → Griesemer、Gerrand       → 并发、goroutine      → 云原生/后端
JavaScript   → John Resig、Mathias      → DOM、V8、性能        → Web 前端
Node.js      → Isaac、TJ                → npm、Express         → 后端/全栈
前端框架      → Rich Harris             → 编译器思想（Svelte）  → Web 全栈
版本控制      → Scott Chacon            → Git 原理             → 团队协作
内容管理      → Matt Mullenweg          → WordPress/CMS        → 建站
虚拟化/音视频  → Fabrice Bellard         → FFmpeg/QEMU          → 音视频/虚拟化
AI 编程       → Thomas Dohmke、Xinchen  → Copilot             → AI 辅助开发
```

**用法**：想深入哪条路线，就去 GitHub 读对应人物的代表仓库源码——比看教程更接近一手。
