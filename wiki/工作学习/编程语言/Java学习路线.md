## 二、AI 模块：今年的突破口

Java 程序员切入 AI 应用开发的具体方向：

- **Spring AI** — Java 生态的大模型应用框架
- **LangChain** — 大模型应用编排
- **Prompt 工程** — 提示词设计
- **RAG** — 检索增强生成
- **Function Calling** — 函数调用
- **MCP** — 模型上下文协议
- **大模型部署** — 把模型跑起来
- **大模型微调** — 定制化训练

> 注：原视频里"RAG Function""MCP 大模型部署""大模型微调转"是语音转写的粘连，此处已拆分规范。
