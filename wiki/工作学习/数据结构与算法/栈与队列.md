# 栈与队列

最常考两个：有效的括号（栈）和滑动窗口最大值（单调队列）。

## 栈(Stack)：后进先出

Python 直接用 list 模拟栈：

```python
stack = []
stack.append(1)   # push
stack.append(2)
top = stack[-1]   # peek → 2
val = stack.pop() # pop → 2
```

## 真题 1：有效的括号

LeetCode 20。面试题里最简单的，但写不出来的大有人在。

**题目**：`s = "()[]{}"` → `true`，`s = "(]"` → `false`

```python
def is_valid(s: str) -> bool:
    pairs = {")": "(", "]": "[", "}": "{"}
    stack = []

    for ch in s:
        if ch in "([{":
            stack.append(ch)       # 左括号入栈
        else:                      # 右括号
            if not stack:           # 栈为空，没有能配对的
                return False
            if stack.pop() != pairs[ch]:  # 不匹配
                return False

    return len(stack) == 0  # 栈必须为空（所有括号都配对了）

# 验证
print(is_valid("()[]{}"))  # True
print(is_valid("([)]"))    # False — 嵌套顺序不对
```

**面试话术**："遇到左括号入栈，遇到右括号检查栈顶是否匹配。最后栈为空说明全部匹配成功。时间 O(n)，空间 O(n)。"

## 真题 2：每日温度

LeetCode 739。考的是**单调栈**——面试如果没刷过大概率想不到。

**题目**：`temperatures = [73, 74, 75, 71, 69, 72, 76, 73]`，输出每个温度要等几天才能升温。
输出 `[1, 1, 4, 2, 1, 1, 0, 0]`

```python
def daily_temperatures(temps):
    n = len(temps)
    answer = [0] * n
    stack = []  # 存元素索引，自底向上温度递减（单调递减栈）

    for i, temp in enumerate(temps):
        # 当前温度 > 栈顶温度 → 栈顶的答案找到了
        while stack and temp > temps[stack[-1]]:
            prev_day = stack.pop()
            answer[prev_day] = i - prev_day
        stack.append(i)

    return answer

# 验证
temps = [73, 74, 75, 71, 69, 72, 76, 73]
print(daily_temperatures(temps))  # [1, 1, 4, 2, 1, 1, 0, 0]
```

**模拟过程**（理解了才能讲给面试官）：
```
i=0, temp=73: stack=[0], answer=[0,0,0,0,0,0,0,0]
i=1, temp=74: 74>73 → pop 0, answer[0]=1, stack=[1]
i=2, temp=75: 75>74 → pop 1, answer[1]=1, stack=[2]
i=3, temp=71: 71<75 → stack=[2,3]
i=4, temp=69: 69<71 → stack=[2,3,4]
i=5, temp=72: 72>69 → pop 4, answer[4]=1
              72>71 → pop 3, answer[3]=2
              72<75 → stack=[2,5]
i=6, temp=76: 76>72 → pop 5, answer[5]=1
              76>75 → pop 2, answer[2]=4
              stack=[6]
i=7, temp=73: 73<76 → stack=[6,7]
```

**单调栈的本质**：维护一个"还没找到答案的元素"的等待队列，一旦遇到更大的值，等待队列里的就找到了答案。

## 队列(Queue)：先进先出

```python
from collections import deque

q = deque()
q.append(1)      # 入队
q.append(2)
first = q[0]     # 看队首 → 1
val = q.popleft() # 出队 → 1
```

**用 deque 不用 list**：`list.pop(0)` 是 O(n)（所有元素向前移动），`deque.popleft()` 是 O(1)。

## 真题 3：用栈实现队列

LeetCode 232。面试爱考这种数据结构之间的模拟。

```python
class MyQueue:
    def __init__(self):
        self.in_stack = []   # 入队栈
        self.out_stack = []  # 出队栈

    def push(self, x):
        self.in_stack.append(x)

    def pop(self):
        self._transfer()  # 如果 out_stack 为空，把 in_stack 倒过来
        return self.out_stack.pop()

    def peek(self):
        self._transfer()
        return self.out_stack[-1]

    def empty(self):
        return not self.in_stack and not self.out_stack

    def _transfer(self):
        """关键：只在 out_stack 为空时才倒，均摊 O(1)"""
        if not self.out_stack:
            while self.in_stack:
                self.out_stack.append(self.in_stack.pop())
```

**均摊 O(1) 分析**：每个元素最多被 push 一次、从 in_stack 移到 out_stack 一次、pop 一次。总共 3n 次操作 / n 个元素 = O(1) 均摊。

## 栈/队列 vs 实际业务

| 数据结构 | 真实场景 |
|----------|---------|
| 栈 | 浏览器后退按钮、函数调用栈、撤销操作 |
| 队列 | 消息队列、BFS 遍历、打印机任务调度 |
| 单调栈 | 股票价格跨度、每日温度、矩形最大面积 |
| 单调队列 | 滑动窗口最大值（TCP 流量控制本质也是这个） |
