# 模型评估与调参

面试必问：**过拟合 vs 欠拟合，交叉验证，准确率/精确率/召回率的区别。**

## 一、过拟合 vs 欠拟合

```python
# 用多项式回归演示
from sklearn.preprocessing import PolynomialFeatures
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error
import numpy as np

# 真实关系: y = sin(x)，加一点噪声
np.random.seed(42)
X = np.linspace(0, 2*np.pi, 20).reshape(-1, 1)
y = np.sin(X).ravel() + np.random.randn(20) * 0.1

# 欠拟合：用线性模型拟合曲线 → 训练集和测试集都很差
model_under = LinearRegression()
model_under.fit(X, y)  # MSE ≈ 0.5（高偏差）

# 刚刚好：用 degree=3 多项式
X_poly3 = PolynomialFeatures(degree=3).fit_transform(X)
model_good = LinearRegression()
model_good.fit(X_poly3, y)  # MSE ≈ 0.01

# 过拟合：用 degree=19 多项式（20 个点 20 个参数 → 完美拟合每个点）
X_poly19 = PolynomialFeatures(degree=19).fit_transform(X)
model_over = LinearRegression()
model_over.fit(X_poly19, y)  # 训练 MSE ≈ 0，但测试集会很差
```

| | 偏差(Bias) | 方差(Variance) | 训练误差 | 测试误差 |
|------|------|------|------|------|
| 欠拟合 | 高 | 低 | 高 | 高 |
| 刚刚好 | 中 | 中 | 低 | 低 |
| 过拟合 | 低 | 高 | 极低 | 高 |

**面试话术**："欠拟合是模型太简单没学到规律，过拟合是把噪声也学了。判断方法：训练误差远小于测试误差就是过拟合；两个都高就是欠拟合。"

## 二、交叉验证（Cross-Validation）

**为什么不能只分一次训练/测试集？** 可能运气好分到简单数据，看起来准确率高，其实泛化差。

```python
from sklearn.model_selection import cross_val_score, KFold
from sklearn.ensemble import RandomForestClassifier

model = RandomForestClassifier()

# 5 折交叉验证
scores = cross_val_score(model, X, y, cv=5, scoring="accuracy")
print(f"各折: {scores}")
print(f"平均: {scores.mean():.3f} ± {scores.std():.3f}")
# 输出: 各折: [0.88 0.92 0.85 0.90 0.87]
#       平均: 0.884 ± 0.024
```

标准差小 = 模型稳定。如果某折特别低，说明那折数据有特殊情况。

## 三、混淆矩阵与分类指标

```
            预测正    预测负
实际正      TP        FN
实际负      FP        TN
```

```python
from sklearn.metrics import (
    accuracy_score, precision_score,
    recall_score, f1_score, confusion_matrix
)

y_true = [0, 1, 0, 1, 0, 1, 0, 0, 1, 1]
y_pred = [0, 1, 1, 1, 0, 0, 0, 0, 1, 1]

cm = confusion_matrix(y_true, y_pred)
# [[4, 1],    TP=3, FN=1
#  [1, 4]]    FP=1, TN=4

print(f"准确率 Accuracy: {accuracy_score(y_true, y_pred):.2f}")    # (4+4)/10 = 0.8
print(f"精确率 Precision: {precision_score(y_true, y_pred):.2f}") # 3/(3+1) = 0.75
print(f"召回率 Recall: {recall_score(y_true, y_pred):.2f}")      # 3/(3+1) = 0.75
print(f"F1: {f1_score(y_true, y_pred):.2f}")                     # 调和平均 = 0.75
```

**面试必问：准确率 vs 精确率 vs 召回率**

| 指标 | 公式 | 问题 | 场景 |
|------|------|------|------|
| Accuracy | (TP+TN)/全部 | 正负样本不均时无用 | 均衡数据集 |
| Precision | TP/(TP+FP) | 预测为正的里面几个真的？ | 垃圾邮件检测（不想误删正常邮件） |
| Recall | TP/(TP+FN) | 真正的正例找到几个？ | 癌症检测（宁可误诊也不能漏诊） |
| F1 | 2×P×R/(P+R) | Precision 和 Recall 的调和平均 | 两者都重要时 |

**面试话术**："精确率和召回率是跷跷板。提高阈值 → 预测更保守 → Precision 升 Recall 降。降低阈值 → 预测更激进 → Recall 升 Precision 降。选哪个指标取决于业务——误报的代价大还是漏报的代价大。"

## 五、面试完整框架

> "你怎么评估一个模型？"

1. **先看数据划分**：训练/验证/测试集分层抽样
2. **选对指标**：分类用 F1 而非准确率（尤其样本不均衡时），回归用 MAE/MSE
3. **交叉验证**：至少 5 折，看均值和标准差
4. **对比基线**：比随机猜好多少？比简单规则好多少？
5. **过拟合检查**：训练集和验证集差距是否过大？
