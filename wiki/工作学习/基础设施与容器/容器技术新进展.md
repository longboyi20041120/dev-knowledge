# 容器技术新进展

## Podman v6.0.0

Docker 的开源替代品重大更新：
- 无守护进程（daemonless）架构，不需要 root 权限
- Pod 概念原生支持（兼容 Kubernetes Pod）
- v6.0 重点改进网络和 systemd 集成
- 与 Docker CLI 高度兼容（`alias docker=podman` 基本可用）

来源：[Podman v6.0.0 公告](https://blog.podman.io/2026/07/introducing-podman-v6-0-0/)

## AWS MicroVMs

Lambda 底层引入 MicroVM：
- 基于 Firecracker，提供完整生命周期控制的隔离沙箱
- 比容器更安全（硬件虚拟化级别隔离），比传统 VM 更快（毫秒启动）
- Serverless 未来的基础设施方向

来源：[AWS Lambda introduces MicroVMs](https://aws.amazon.com/blogs/aws/run-isolated-sandboxes-with-full-lifecycle-control-aws-lambda-introduces-microvms/)
