# Vivo 手机 AI 操控助手

## 一句话

一个可以直接操控手机桌面的 AI Agent——比如自动把 100 个 App 智能分类到文件夹、清理无用通知、优化系统设置。

## 背景

手机会越用越乱，软件散落在十几页桌面，通知越来越多，系统设置越改越卡。用户自己整理太累，现有的手机助手（Vivo Jovi 等）只能做简单任务，无法真正"操控手机"。

## 核心场景

- **自动整理桌面**：AI 扫描所有 App → 按类型/用途自动分类到文件夹（"社交"、"购物"、"工具"等）
- **通知管理**：分析通知重要性，自动屏蔽低频无用的
- **垃圾清理**：比现有清理工具更智能，知道什么能删什么不能
- **系统优化**：根据使用习惯调整电池策略、后台限制

## 可行性

### 技术上

- Android **AccessibilityService** API 可以做到：模拟点击、滑动、读取屏幕内容、打开应用
- 不需要 root 权限
- 需要用户手动开启无障碍服务
- Vivo 的 Funtouch OS 可能对后台服务有限制（需要加入白名单）

### 技术栈建议

```
Android App (Kotlin/Java)
├── AccessibilityService  ← 操控手机的核心
├── UsageStatsManager     ← 获取 App 使用频率
├── PackageManager        ← 获取已安装 App 信息
└── 本地 LLM 或 API       ← 智能分类决策
```

### 挑战

| 挑战 | 严重度 | 解决思路 |
|------|--------|----------|
| 无障碍权限用户不敢开 | 高 | 开源 + 隐私说明，本地运行不传数据 |
| Vivo 杀后台 | 中 | 引导用户加白名单 |
| 不同手机桌面布局不统一 | 高 | 先聚焦 Vivo，再适配其他 |
| Google Play 政策 | 中 | 无障碍服务需声明用途 |

## 类似产品

- Vivo Jovi：只能语音问答和简单设置，不能操控桌面
- Tasker：自动化强大但学习成本高，需要手动编程
- 小米 AI 助手：类似 Jovi，功能有限
