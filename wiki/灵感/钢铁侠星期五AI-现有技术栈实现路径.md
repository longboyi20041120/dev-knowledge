# 钢铁侠星期五 AI — 现有技术栈实现路径

## 一句话

用今天已有的技术去逼近电影里的星期五（FRIDAY），能拼出一个「70 分的躯壳」——多模态大模型当脑子、Agent 框架当神经、向量库当记忆、机器人 + API 当手脚；但最后 30 分（真自主、真可靠、真灵巧、真自我）恰恰是从「专才」走向 AGI 还没跨过的坎。

## 背景

聊到「星期五和现在的 AI 有什么不同」时展开的：星期五是科幻里的**全能通用 AI**（有感官、有手脚、有自我、无所不能），现在的 AI 是现实里的**专才**（语言知识很强，但看不见、摸不着、没自我、各自为战）。这篇把「如果现在就要造一个」的技术栈拆开记下来。关联 [[工作学习/AI底层原理|AI 底层原理]] 里的模型基础。

## 星期五拆成五个能力模块

把「一个全能 AI」拆成可映射到现实技术的五块：

| 模块 | 作用 | 成熟度 |
|------|------|--------|
| 大脑 | 理解意图 + 推理 + 规划 + 生成 | 最成熟 |
| 感官 | 看得见、听得见、会说话 | 进步最快 |
| 手脚 | 真的动手（物理 + 数字） | 最不成熟、最难 |
| 记忆 | 长期记忆 + 个性化人格 | 较成熟 |
| 编排 | 把上面四块串成一个整体 | 最近最热 |

## 一、大脑（推理与决策核心）

- **底座**：多模态大语言模型（GPT / Claude / Gemini 类），负责理解、推理、规划、语言生成。
- **关键范式**：**Agent 智能体架构**。光有模型不够，要让它「想 → 调工具 → 看结果 → 再想 → 再做」自主循环。这就是 [[前沿信息/Agent-Skill生态|Agent Skill 生态]] 里说的 Agentic AI，Claude Code 就是个简化版。星期五 = 这个模式放大 1000 倍。

## 二、感官（多模态感知）

- **视觉**：实时视频理解模型、YOLO 类目标检测、SLAM（空间定位建图）
- **听觉**：Whisper 类语音识别 + 声纹 + 环境声分析
- **说话**：实时语音合成（TTS，ElevenLabs 那种以假乱真的）
- **融合**：多模态大模型把「看到 + 听到 + 说」统一进一个大脑

## 三、手脚（具身 / 执行）——最难的一环

- **物理世界**：机器人本体（波士顿动力类）+ 机械臂 + 具身智能模型（RT-2、Figure 机器人）
- **数字世界**：调用 API、操作电脑（Computer Use）、控制 IoT 智能家居
- 电影里星期五开战衣、调无人机 → 现实对应「具身智能 + 设备控制协议」

## 四、记忆（长期记忆与人格）

- **向量数据库**（Pinecone / Milvus）：存长期记忆，随时检索
- **RAG 检索增强**：把记忆喂回大脑，让它「记得你」
- **持续学习 / 微调**：形成稳定的性格与偏好

## 五、编排层（贯穿全程的神经系统）

- **Agent 编排框架**：LangChain / LlamaIndex / AutoGPT 类，负责任务分解、工具调度、多步规划
- **多智能体协作**：一个「主星期五」指挥一堆子专家（飞行 / 医疗 / 黑客），分工又统一

## 架构总览

```
          ┌─────────────────────────────┐
          │   感官层  视觉/听觉/语音       │  ← 多模态模型
          └──────────────┬──────────────┘
                         │ 感知输入
          ┌──────────────▼──────────────┐
          │   大脑  多模态LLM + Agent推理  │  ← 核心决策
          │        ↕ 记忆(向量库/RAG)      │
          └──────────────┬──────────────┘
                         │ 决策输出
          ┌──────────────▼──────────────┐
          │  执行层  API调用/机器人/IoT    │  ← 具身+设备控制
          └─────────────────────────────┘
                  ↑ 编排框架贯穿全程 ↑
```

## 可行性

- **技术可行**：每一块都有现成技术，但拼起来会露馅。
- **五道坎**：
  1. **延迟**——多模态 + 推理 + 执行串一圈，慢得肉眼可见，做不到星期五的即时反应
  2. **可靠性**——大模型会幻觉，战衣控制这种生死攸关的事不敢全交给它（呼应 [[前沿信息/最少AI原则|最少 AI 原则]]）
  3. **具身太弱**——机器人手脚还笨，离电影里的灵巧差得远
  4. **没有真自主**——现在 Agent 是被动响应，星期五那种主动关心、主动预判本质是 AGI
  5. **算力成本**——24 小时全感知全推理，烧的算力是天文数字（相关 [[工作学习/AI-Agent-Token优化|AI Agent Token 优化]]）

## 关键判断

有意思的是，**编排层（Agent）是最近两年最热的方向**。很多人赌的是：不一定非要等一个超级大脑，把一堆专才用足够聪明的方式**编排**起来，可能就能先攒出一个「够用的星期五」。这也是为什么现在 Agent / 多智能体框架这么火。
