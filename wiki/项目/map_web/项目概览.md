# 项目概览 — 智慧校园三维导览系统

## 一句话

一个基于 Three.js 3D 地图的校园导航系统，帮助学生和访客快速找到建筑位置和路径。

## 技术栈

| 层 | 技术 | 说明 |
|-----|------|------|
| 前端 | Three.js r166 | 3D 场景渲染，GLB 模型加载，ES Module via CDN |
| 前端文件 | 单文件 `1.html` | ~3500 行，CSS + HTML + JS 合一 |
| 后端 | Spring Boot 3.x | REST API，端口 8080 |
| ORM | MyBatis-Plus | 数据库映射 |
| AI 服务 | Python FastAPI | 端口 8000，通过 Java 反向代理 |
| 大模型 | Ollama + qwen2.5:7b | 本地运行，端口 11434 |
| 数据库 | MySQL | `smart_campus` 库，utf8mb4 |
| 3D 模型 | SketchUp → GLB | `campus.glb`，~2.1MB，89 栋建筑 |

## 架构

```
浏览器 (1.html, Three.js)
    ↓ AJAX
Java :8080 (Spring Boot, REST API)
    ├── /api/ai/* → 反向代理 → Python :8000 (FastAPI) → Ollama :11434
    ├── /api/weather/* → 和风天气 + Open-Meteo
    └── /api/buildings/*  /api/path/* → MySQL :3306
```

## 关键设计决策

- **单文件前端**：所有 UI 在 `1.html` 里，CSS/JS 内联
- **modelId 匹配**：前端通过建筑名与 GLB 节点关联
- **AI 反向代理**：前端只访问 :8080，AI 请求由 Java 转发到 Python
- **A* 导航**：156 个路点，红色路径线
- **UI 风格**：浅色玻璃拟态，墨色单主色系统

## 核心文件

| 文件 | 行数 | 职责 |
|------|------|------|
| `1.html` | ~3500 | 全部前端 |
| `PathServiceImpl.java` | ~200 | A* 算法 + 校园导航 |
| `llm_service.py` | ~300 | AI 对话 + NAV 标记解析 |
| `AiProxyController.java` | ~30 | 反向代理 |

## 启动

```bash
# 1. 确保 MySQL + Ollama 运行
# 2. AI 服务
cd ai-service && python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
# 3. 后端
mvn spring-boot:run
# 4. 浏览器 → http://localhost:8080/1.html
```
