# 前端架构 — map_web

## 技术栈

- Three.js r166 (ES Module, jsdelivr CDN)
- OrbitControls, GLTFLoader, EffectComposer, RenderPass, OutlinePass, OutputPass
- 单文件架构：`1.html` (~4500 行，CSS+HTML+JS)

## UI 布局（四角 HUD + 底栏）

| 位置 | 组件 | 功能 |
|------|------|------|
| 左上 | 建筑目录面板 | 展开悬浮面板，分类筛选，可拖动，≤700px 可折叠 |
| 中上 | 搜索栏 | 建筑搜索，实时下拉建议 |
| 右上 | 天气面板 | 温度 + AQI + 生活指数，可折叠 |
| 左下 | 课程按钮 | 展开悬浮面板，课表/列表双视图 |
| 中下 | 导航栏 | 起点/终点选择，路点采集，导航模式 |
| 右下 | AI 悬浮球 | 精灵图动画宠物，可拖动拖拽，点击展开对话 |

## 响应式布局（demo_8 重构）

所有面板使用 CSS 自定义属性 + `clamp()` 平滑缩放，位置相对计算。

```css
--g:        clamp(6px, 0.78vw, 14px);          /* 全局间距 */
--left-w:   clamp(220px, calc(100px + 11.46vw), 340px);
--right-w:  clamp(180px, calc(60px + 9.38vw), 260px);
--search-w: clamp(240px, calc(120px + 12.50vw), 380px);
```

**断点策略**：
| 宽度 | 行为 |
|------|------|
| >1300px | 全布局，建筑信息贴左面板右侧 |
| ≤1300px | 建筑信息变为居中浮动卡片，放在搜索栏下方 |
| ≤900px | 右侧天气面板可折叠 |
| ≤700px | 左侧建筑目录可折叠，导航栏全宽 |

**3D 相机**：正交相机，**水平视野固定 2000 单位**，resize 时重设 pixelRatio 防止降分辨率。

## 设计系统

**主题**：浅色玻璃拟态 (glassmorphism)

```css
--bg: #faf8f5;
--panel: rgba(255,255,255, 0.94);
--panel-border: rgba(0,0,0, 0.06);
--accent: #2563eb;
```
--accent: #3568e5;
```

**规则**：
- 单主色系统（只用 `--accent` 蓝色强调）
- 灰度深浅区分层级，不引入新颜色
- 10px-14px 圆角，按钮/输入框 12px
- backdrop-filter blur 为浮层提供空间感
- 交互反馈用 background-color 切换 (150ms)，不用 transform

## 关键函数

| 函数 | 用途 |
|------|------|
| `showToast(msg, type)` | 通知提示，3s 自动消失 |
| `getToken()` / `authHeaders()` | JWT 认证 |
| `loadBuildingTypeContent(buildingId)` | 加载建筑服务+食堂+课程 |
| `openBuildingByClassroom(roomName)` | 从课程教室打开建筑 |
| `h()` | HTML 转义，防 XSS |

## 建筑分类

分类含逗号（如 `"宿舍,食堂"`）表示同时属于多个分类。`campus_building.alias` 存俗称，前端显示 `别名（正式名）`。

## 导航模式

状态机：`idle → selecting_start → selecting_end → navigating`

- 路线：起点建筑底面 → 最近路点 → A* 路点序列 → 终点最近路点 → 终点建筑底面
- 路线用红色 THREE.Line 绘制

## 3D 场景配置

| 属性 | 值 |

| 建筑材质色 | `#fcfdff` |

| 环境光强度 | 0.95 |
| 建筑选中动画 | 弹簧压缩 (100ms squash → 回弹) |
