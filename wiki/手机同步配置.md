# 手机端 Obsidian 同步指南

## 架构

```
桌面 Obsidian → GitHub → 手机下载 ZIP → 手机 Obsidian
```

## 初始配置

1. 手机浏览器打开 `https://github.com/longboyi20041120/dev-knowledge`
2. 绿色 **Code** 按钮 → **Download ZIP**
3. 解压 ZIP（注意：**必须确保解压出 `.obsidian` 隐藏文件夹**）：
   - **方法一**：vivo 自带文件管理器 → 打开"显示隐藏文件" → 直接解压
   - **方法二**：用 ZArchiver → 设置里打开"显示隐藏文件" → 解压到 `内部存储/Documents/devkb`
4. 确认 `Documents/devkb/.obsidian` 文件夹存在
5. Obsidian → **打开文件夹作为仓库** → 选 `Documents/devkb`

## 日常更新

知识库有更新时，重复上面步骤 1-4，用 ZArchiver 把新 ZIP 覆盖到 `Documents/devkb` 就行。

> ⚠️ **无论用什么工具解压，关键一步：打开"显示隐藏文件"**。点号开头的 `.obsidian` 文件夹默认被隐藏，不开就丢失图视图颜色分组、插件配置等。详见 [[踩坑记录/Android解压丢失obsidian隐藏文件夹]]。
