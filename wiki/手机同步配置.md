# 手机端 Obsidian 同步指南

## 架构

```
桌面 Obsidian → GitHub → 手机下载 ZIP → 手机 Obsidian
```

## 初始配置

1. 手机浏览器打开 `https://github.com/longboyi20041120/dev-knowledge`
2. 绿色 **Code** 按钮 → **Download ZIP**
3. **用 ZArchiver 解压**（vivo 自带文件管理器解压会丢失 `.obsidian` 隐藏文件夹）：
   - ZArchiver 右上角菜单 → 设置 → **打开"显示隐藏文件"**
   - 找到 `Download/dev-knowledge-main.zip` → 解压到 `内部存储/Documents/devkb`
4. 确认 `Documents/devkb/.obsidian` 文件夹存在
5. Obsidian → **打开文件夹作为仓库** → 选 `Documents/devkb`

## 日常更新

知识库有更新时，重复上面步骤 1-4，用 ZArchiver 把新 ZIP 覆盖到 `Documents/devkb` 就行。

> ⚠️ **不要用手机自带的文件管理器解压**，它不处理 `.obsidian` 这类点号开头的隐藏文件夹，会导致图视图颜色分组、插件配置等全部丢失。详见 [[踩坑记录/Android解压丢失obsidian隐藏文件夹]]。
