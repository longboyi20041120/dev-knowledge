# 手机端 Obsidian 同步指南

## 架构

```
桌面 Obsidian → GitHub → 手机下载 ZIP → 手机 Obsidian
```

## 初始配置

1. 手机浏览器打开 `https://github.com/longboyi20041120/dev-knowledge`
2. 绿色 **Code** 按钮 → **Download ZIP**
3. 文件管理器 → `Download` → 解压 `dev-knowledge-main.zip`
4. 重命名解压出的文件夹为 `devkb`
5. 移动到 `内部存储/Documents/devkb`
6. Obsidian → **打开文件夹作为仓库** → 选 `Documents/devkb`

## 日常更新

知识库有更新时，重复上面步骤 1-4，把新 ZIP 覆盖到 `Documents/devkb` 就行。
