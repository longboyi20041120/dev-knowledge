# Hermes 管理员快捷方式 · 操作指南

目的：让 Hermes 以管理员权限启动，减少手机远程时被系统（UAC）拦截。**这几步需你在真机上点击完成**（Claude Code 隔着环境做不了）。相关：[[AI协作交接板]]。

## 第一步：做启动脚本

1. 桌面右键 → 新建 → **文本文档**，打开它。
2. 粘贴以下内容（**可直接复制**）：

```bat
@echo off
cd /d C:\Users\14778\hermes-agent
hermes
```

> ⚠️ 最后那行 `hermes` 换成**你平时启动 Hermes 做微信控制时实际敲的命令**。若平时就是 `hermes`，不用改；若是 `hermes gateway` / `hermes serve` 或带参数，换成你那条。

3. 文件 → 另存为：
   - 文件名：`启动Hermes.bat`（**结尾必须是 `.bat`，不是 `.txt`**）
   - 保存类型：**所有文件**
   - 编码：**ANSI**
   - 存到桌面

## 第二步：做快捷方式并设为管理员

1. 右键 `启动Hermes.bat` → **创建快捷方式**（桌面多出"启动Hermes - 快捷方式"）。
2. 右键这个**快捷方式** → **属性**。
3. "快捷方式"标签页 → 右下角 **高级(A)...** 按钮。
4. 勾选 **"用管理员身份运行(R)"** → 确定。
5. 应用 → 确定。

**完成**：以后双击这个快捷方式，Hermes 会以管理员启动（弹一次 UAC 点"是"）。

## 提醒

- ⚠️ **先关掉旧的 Hermes 再用它启动**，别让两个实例同时跑（会抢写/打架）。
- ⚠️ **管理员 = 放开手脚也放大风险**：远程误操作后果更大。配合"危险操作让 Hermes 发微信问你确认"更稳（此机制的具体配置待下次落实）。
