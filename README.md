# 编程知识库

欢迎。这是你的 AI 编程知识库。

## 怎么用

1. **扔原材料**：把网页文章、报错日志、灵感碎片扔进 `raw/`
2. **AI 处理**：AI 自动读取、整理、写入 `wiki/`
3. **你在 Obsidian 里看**：浏览、搜索、Graph View 发现关联

## 快速入口

- [[wiki/索引|知识库索引]] — 所有笔记的目录
- [[wiki/最近更新|最近更新]] — 最近改了什么
- `raw/` — 扔东西的地方
- `模板/` — 笔记格式模板
